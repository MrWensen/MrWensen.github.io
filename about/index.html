About

In the summer vacation of 2021, I started to learn how to build a personal blog. In the beginning, I learnt how to deploy a blog from Bilibili. And then,I successfully deployed my blog on Github.

But for some reason, the network condition is not good, so I made up my mind to put my blog on the server in China. On Nov 11, I bought a server and a domain name. And continue to learn how to deploy blogs to servers, Be aware that blogging on a server is different from blogging on Github, especially for a rookie.

I finished a PPT course in 100 days in 2022, I started updating notes on my blog.

At the same time, I constantly polished the article. In the future, I plan to update some other study notes and some software I have used.

If you like my content, welcome to interact with me.
